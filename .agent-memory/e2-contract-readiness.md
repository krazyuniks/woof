# E2 Contract Readiness Notes

Durable notes for Woof E2 as of 2026-06-10.

- S1 + S2 are shipped on `main`. S1 is the deterministic Stage-2.5 readiness seam: `NodeType.CONTRACT_READINESS`, `next_node` routing after `definition_closed`, `readiness-result.schema.json`, and `readiness_gate` schema/event/write support. S2 is the full check matrix in `src/woof/graph/readiness.py`.
- The six checks: `readiness_acceptance_signal` (satisfied by a non-deprecated contract decision related to the outcome, or an acceptance criterion that names it with a concrete signal - a bare `O<n>`/`CD<n>` mention is not a signal), `readiness_acceptance_prose` (subjective prose blocks unless paired with a concrete signal), `readiness_contract_concreteness` (non-deprecated CDs carry a non-placeholder ref unless forward-created), `readiness_path_resolution` (openapi/json-schema file parts and backtick body/notes paths resolve against `git ls-files`), `readiness_symbol_resolution` (file-based `file.py:Symbol` refs need a tracked file plus a top-level `ast` class/function; module/dynamic refs require a forward-created marker), and `readiness_decomposition_sufficiency`.
- Forward-created grammar is exactly `` `tok` (forward-created) `` or `` `tok` (created by ticket <id>) ``, parsed only from the EPIC body and contract-decision notes. Malformed annotations exempt nothing. openapi/json-schema refs match on file part; pydantic refs match the full `file.py:Symbol` and the bare file token.
- `evaluate_readiness(repo_root, epic_id, epic_path)` keeps the three positional args; `time_budget_s` is keyword-only. Budget exhaustion skips path/symbol resolution and emits one `readiness_checker_budget` warn (ok true) that never pulls `ReadinessResult.ok` false and never opens a gate on its own. Tests force this with `time_budget_s=0`.
- S3 is next and not yet implemented: readiness recycle escalation and audited readiness-gate resolutions (`revise_epic_contract`, `approve_with_reason`, `abandon_epic`). Do not assume readiness-gate resolution semantics beyond the generic gate record until S3 lands.
