# Efficiency Runs

This directory is for redacted EF-007 run manifests and comparison tables that
are intentionally kept as evidence.

Allowed here:

- redacted JSON manifests produced by `just efficiency-bench run`;
- Markdown comparison tables produced by `--compare`;
- short notes explaining the run context and quality decision.

Do not store raw consumer `.woof/epics/E*/audit/` output, worktrees, model
transcripts, credentials, or unredacted logs in this directory.
